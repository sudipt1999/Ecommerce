### Setting Up The Environment

**Prerequisites**
1. npm and node should be installed<br>
2. mongo db should be installed<br>

#### For Backend Server
**Step 1:** `Clone the repo` <br>
**Step 2:** `cd Ecommerce`    <br>
**Step 3:** `npm install`      <br>
 
#### For FrontEnd 
**Step 1:** `cd ui` <br>
**Step 2:** `npm install` <br>
